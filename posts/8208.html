量子力学有关的一道作业题
justaLoli
  2023-09-20 21:07:09 2023-09-20 21:07   2023-10-31 12:25:51  
 
 988 字  5 分钟

波函数：

\[ \Psi(x,t) = \Psi_0 e^{\frac{i}\hbar (p_xx-Et)} \]

其中：

\[ p_x = mv = \frac{h}\lambda 为粒子动量；\]

\[ E = mc^2 = h\nu 为粒子能量；\]

\[ \hbar = \frac{h}{2\pi}\]

note: 可以类比机械波方程：

\[ y(x,t) = A\cos(kx - \omega t) \]

其中：

\[ k = \frac{2\pi}\lambda 为波数 \]

对波函数微分：

\[ \begin{equation}\label{解释E} \frac{\partial\Psi(x,t)}{\partial t} = -\frac{iE}\hbar \Psi(x,t) \end{equation}\]

\[ \frac{\partial^2\Psi(x,t)}{\partial x^2} = -\frac{p_x^2}{\hbar^2} \Psi(x,t) \]

由\(E = \frac{p_x^2}{2m}\)（动量和能量的关系），得：

\[ i\hbar\frac\partial{\partial t}\Psi(x,t) = -\frac{\hbar^2}{2m}\cdot\frac{\partial^2}{\partial x^2}\Psi(x,t) \]

这是一维运动自由粒子含时的薛定谔方程。

对于处在势能场中的粒子，\(E = \frac{p_x^2}{2m} + U(x,t)\)，上式变为：

\[ i\hbar\frac\partial{\partial t}\Psi(x,t) = -\frac{\hbar^2}{2m}\cdot\frac{\partial^2}{\partial x^2}\Psi(x,t) + U(x,t)\Psi(x,t) \]

在三维势场中：

\[ i\hbar\frac{\partial\Psi}{\partial t} = H\Psi \]

其中：

\[ H = -\frac{\hbar^2}{2m}\nabla^2 + U(\vec{r},t) \]

如果势场不随时间变化，则U与t无关，从而薛定谔方程中，\(\vec{r}\)与\(t\)为独立变量，可以用变量分离法求解：

\[ \Psi(\vec{r},t) = \psi(\vec{r})f(t) \]

代入薛定谔方程：

\[ \left[-\frac{\hbar^2}{2m}\nabla^2\psi(\vec r) + U(\vec r )\psi(\vec r )\right]\frac1{\psi(\vec r)} = i\hbar\frac{\partial f(t)}{\partial t} \frac1{f(t)} = E\]

为什么等于能量E？

上文 \((\ref{解释E})\) 中，对波函数微分：

\[ \frac{\partial\Psi(x,t)}{\partial t} = -\frac{iE}\hbar \Psi(x,t) \]

能得到

\[i\hbar\frac{\partial\Psi}{\partial t}\cdot \frac1\Psi = E\]

可以验证变量分离后的等式等于E。

  1. 三维无限深势阱中电子的能级和波函数。

势阱：

\[ U = \left\{\begin{array}{rcl} 0 & & {0 < x < a},{0<y<a},{0<z<a} \\ \infty & & 其他情况 \end{array}\right. \]

薛定谔方程：

\[ i\hbar\frac{\partial\Psi}{\partial t} = H\Psi \]

定态，做变量分离：

\[ \Psi(\vec{r},t) = \psi(\vec{r})f(t) \]

\[ \left[-\frac{\hbar^2}{2m}\nabla^2\psi(\vec r) + U(\vec r )\psi(\vec r )\right]\frac1{\psi(\vec r)} = i\hbar\frac{\partial f(t)}{\partial t} \frac1{f(t)} = E\]

对于\(f\)，易得：

\[ f(t) = e^{-i\frac E \hbar t} \]

对于\(\psi\)，对于阱外，\(U = \infty\)，能得到\(\psi \equiv 0\)。阱内，U = 0：

\[\nabla^2 \psi(x,y,z) + \frac{2mE}{\hbar^2}\psi(x,y,z) = 0\]

由于体系的对称性，做变量分离：

\[ \psi(x,y,z) = X(x)Y(y)Z(z) \]

同时将E分为三部分\(E_x + E_y + E_z\)，得：

\[X''(x)Y(y)Z(z) + X(x)Y''(y)Z(z) + X(x)Y(y)Z''(z) + \frac{2m(E_x + E_y + E_z)}{\hbar^2}X(x)Y(y)Z(z) =0\]

可以分离得到：

\[\left\{ \begin{aligned} X''(x) + \frac{2mE_x}{\hbar^2}X & = 0 \\ Y''(y) + \frac{2mE_y}{\hbar^2}Y & = 0 \\ Z''(z) + \frac{2mE_z}{\hbar^2}Z & = 0 \end{aligned}\right.\]

\[X(x) = C_x sin(k_xx + \phi)\]

其中

\[ k_x = \sqrt{\frac{2mE_x}{\hbar^2}}\]

利用边界条件确定参数：

\[X(0) = 0 \Rightarrow \phi = n\pi ，为方便直接取\phi = 0。X(x) = C_x sin(k_xx )\]

\[X(a) = 0 \Rightarrow k_xa = n_x\pi，k_x = \frac{n_x\pi}{a}, n = 0,1,2,... \]

所以

\[X(x) = C_x sin(\frac{n_x\pi x}{a})\]

类似的, \[Y(y) = C_y sin(\frac{n_y\pi y}{a})\] \[Z(z) = C_z sin(\frac{n_z\pi z}{a})\]

所以

\[\psi(x,y,z) = C sin(\frac{n_x\pi x}{a})sin(\frac{n_y\pi y}{a})sin(\frac{n_z\pi z}{a})\]

\[\Psi(x,y,z,t) = C sin(\frac{n_x\pi x}{a})sin(\frac{n_y\pi y}{a})sin(\frac{n_z\pi z}{a}) e^{-i\frac E \hbar t} \]

归一化：

\[\begin{aligned} & \int_\tau |\Psi|^2 d\tau \\ = & C^2\int_0^a sin^2(\frac{n_x\pi}{a}x)dx\int_0^a sin^2(\frac{n_y\pi}{a}y)dy\int_0^a sin^2(\frac{n_z\pi}{a}z)dz \\ = & C^2 \frac {a^3}8 = 1 \end{aligned}\]

所以 \[C = \sqrt{\frac8{a^3}}\]

注意到\(k_x = \frac{n_x\pi}{a}\)，而同时 \(k_x = \sqrt{\frac{2mE_x}{\hbar^2}}\) ，所以

\[E_x = \frac{n_x^2\pi^2\hbar^2}{2ma^2}\]

所以 \[E = \frac{(n_x^2+n_y^2+n_z^2)\pi^2\hbar^2}{2ma^2}\] 为该电子的能量（能级）。

\[\Psi(x,y,z,t) = \sqrt{\frac8{a^3}} sin(\frac{n_x\pi x}{a})sin(\frac{n_y\pi y}{a})sin(\frac{n_z\pi z}{a}) e^{-i\frac{(n_x^2+n_y^2+n_z^2)\pi^2\hbar}{2ma^2}t} \]
